# weather-app
PyQt اپلیکیشن ساده وضعیت آب و هوا با پایتون و 

جستجوی وضعیت آب و هوا بر اساس نام شهر (به فارسی یا انگلیسی)
نمایش اطلاعات فعلی:
دما، احساس دما، رطوبت، سرعت باد
وضعیت هوا (با توصیف فارسی)
زمان طلوع و غروب خورشید (با توجه به timezone محلی)
آیکون بزرگ و مرتبط با وضعیت هوا

پیش‌بینی هوشمند ۵ روز آینده:
حداقل/حداکثر دما
آیکون غالب برای هر روز (با اولویت به شرایط مهم مانند باران، برف یا طوفان)
نام روز به فارسی

رابط کاربری زیبا با تم تاریک (dark teal) و آیکون‌های FontAwesome
بارگذاری asynchronous (بدون فریز شدن برنامه)
مدیریت خطاها (شهر یافت نشد، عدم اتصال اینترنت، timeout و غیره)
